<h1 align="center">森和编程 API接口端</h1>

## 首先安装依赖

```bash
$ npm install
```

## 运行这个App

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
